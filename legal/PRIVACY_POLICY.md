2. User content and collection data

3. Image data

4. Public sharing data

5. Billing and subscription data
- Stripe customer/subscription identifiers, plan status, period end dates, and billing state (for example active, past due, canceled).

6. Security and abuse-prevention data
- Captcha signals, authentication logs, and operational security logs.

7. Device and technical data
- Basic request, session, cookie, and app telemetry needed to run and secure the Service.
